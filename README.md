HTTP only GSM/GPRS & GPS Shield Library for modules using SIM900/SIM908
======
This fork has been stripped down to HTTP support in order to save precious memory on your Arduino.

### Install Notes:
Place the files in your arduino/libraries/GSM-GPRS-GPS-Shield folder.
